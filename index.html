2-4 Class Id, Class,Apply styles to mulitple Elements, Id vs class

ID

ID দিয়ে এইচটিএমএল এর একটি নির্দিষ্ট ইলিম্যান্টকে স্টাইল দেওয়ার কাজে ব্যবহৃত হয়। • একটি ইলিম্যান্ট এর জন্য একটি মাত্র ID ব্যবহার করা যায়। • একটি ID একটি পেইজে মাত্র একবার ব্যবহার করা যায়। সিএসএস এ id এর নাম লেখা শুরু করার আগে # চিহ্ন দিতে হয় তারপর id এর নাম এবং শেষে দ্বিতীয় ব্র্যাকেটের মধ্যে সিএসএস রুল গুলো লিখতে

Class

অপর দিকে class দিয়ে একের অধিক ইলিম্যান্টকে স্টাইল দেওয়া যায়। • একই class একের অধিক ইলিম্যান্ট এর মধ্যে থাকতে পারে। • একই ইলিম্যান্ট এর একের অধিক class থাকতে পারে। সিএসএস এ class এর রুল গুলো লেখা শুরু করার আগে .
